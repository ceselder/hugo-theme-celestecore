# celeste-core

The theme I use for [celeste.computer](https://celeste.computer)

A modified version of [nostyleplease](https://github.com/hanwenguo/hugo-theme-nostyleplease)

## Main modifications
* Changed bg color to `#`
* Added [fira code](https://fonts.google.com/specimen/Fira+Code) as the main font
* Added reading time for articles
